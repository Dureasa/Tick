Tick
终端美学 | 简洁高效 | 类 Vim 交互式待办事项管理工具
项目理念
不同于普通命令行 todo 工具，本项目旨在打造一款运行在终端内、拥有独立可视化界面的轻量待办管理软件。
它不依赖复杂桌面环境，却能提供清晰美观的 TUI（终端用户界面），以极简线条勾勒布局，支持键盘交互、多视图切换与时间维度管理，让终端用户在纯文本环境下也能拥有舒适高效的任务管理体验。
核心设计思路
美观简洁的终端界面：使用线条边框划分区域，整体布局干净统一
类 Vim 交互体验：启动后进入专属界面，支持键盘导航，无需频繁输入命令
交互式任务列表：任务以条目形式展示，可上下选择，回车进入详情或操作
时间维度管理：每条待办可绑定时间，支持按日期筛选、按优先级 / 截止时间排序
多视图切换：提供清单视图、日历视图、分类视图等不同展示方式
轻量化无负担：纯终端运行，启动快、操作流畅，适合开发者与日常使用
界面与操作设想
启动后直接进入主界面，由边框构成列表区、状态栏、提示区
上下键 /j/k 选择任务条目，回车键进入任务详情或完成状态切换
支持快捷键切换视图：今日任务、本周任务、全部任务、已完成 / 未完成
新增任务时可选择时间：今日、明日、本周、自定义日期
底部状态栏实时显示任务总数、已完成数、当前视图模式
整体操作逻辑接近 Vim，学习成本低，专注高效
目标
做一款好看、好用、轻量、纯粹的终端待办工具，让终端不再只是执行命令，也能成为优雅的生产力空间。
